Irish Life Fund Analytics Dashboard

An interactive analytics dashboard tracking 136 Irish Life-domiciled funds, built to make fund risk, returns, and ESG classification easy to explore and compare at a glance.

🔗 Live Dashboard: [ajay-wagh-projects.github.io/Irish-Life-Fund-Analytics-Report-](https://ajay-wagh-projects.github.io/Irish-Life-Fund-Analytics-Report-/)

 📊 Overview

This project consolidates and analyzes performance data across the full Irish Life fund range, helping investors and analysts compare funds by risk band, return horizon, and sustainability classification — all from a single interactive interface.

Performance data as of: 27 Feb 2026 | 136 Funds

 🗂️ Dashboard Tabs

 📊 Overview
A portfolio-level snapshot covering:
- Total funds (136) across IL1–IL7 risk bands
- Article 8 (ESG) fund count and % of portfolio
- Risk distribution across all funds
- SFDR classification split (Article 8 vs Article 6)
- Average 1-year return by risk band
- Fund launches per year (range growth over time)
- Risk vs 5-year return scatter plot

 📈 Performance
Deep-dive into returns across time horizons and risk levels:
- Filterable by risk band (IL1–IL7)
- Average returns by period and risk band (1M, 6M, 1Y, 3Y, 5Y, 10Y)
- Risk vs 1-year return scatter
- Risk vs 10-year return scatter (long-term compounding effect)

 🌿 Sustainability (SFDR)
Examines whether ESG classification affects performance:
- Article 8 vs Article 6 fund counts
- Average 1-year returns compared across classifications
- SFDR distribution across risk bands

 📋 All Funds
A full, filterable fund directory with:
- Fund name, risk rating (IL1–IL7), and SFDR classification
- Returns across 1 month, 6 months, 1 year, 3 years, 5 years, and 10 years
- Filters by risk band and SFDR status

 🛠️ Tools & Tech Stack

- Power BI — interactive visuals and dashboard design
- Python (Pandas) — data cleaning, transformation, and fund-level calculations
- Advanced Excel — data preparation and validation
- GitHub Pages — hosting the interactive web version

 💡 Key Insights

- Higher-risk funds (IL bands) generally show higher average 1-year returns, illustrating the expected risk/reward tradeoff
- 42% of the fund range is now ESG-classified (Article 8), reflecting growing sustainability focus in the Irish fund market
- The scatter analyses (risk vs 5Y/10Y return) help surface funds that outperform their risk band over the long term

 🚀 How to Use

Visit the [live dashboard](https://ajay-wagh-projects.github.io/Irish-Life-Fund-Analytics-Report-/) and:
1. Start on the Overview tab for a portfolio-wide snapshot
2. Use the Performance tab to filter by risk band and compare returns across time horizons
3. Check Sustainability to compare ESG vs non-ESG fund performance
4. Browse All Funds for the complete, filterable fund list
